🔧 드래그&드롭 순서 변경 즉시 반영 개선 완료

⚠️ 문제 분석

증상: 드래그&드롭으로 순서 변경 시 화면에 즉시 반영되지 않고 새로고침해야 보임

원인: 순서 변경 후 display_order 필드 기준 정렬이 누락되어 UI가 업데이트되지 않음

✅ 해결 방법

접근: 모든 리스트 표시 지점에서 display_order 기준 정렬 추가

범위: 직원관리, 시설관리 페이지 및 드래그&드롭 컴포넌트

🔨 수정된 파일 및 변경사항

✓
DragDropStaffList.tsx
필터링 및 드래그 완료 후 정렬 로직 추가
수정 전:
const filteredStaff = staffMembers.filter(staff => { if (selectedCategory === 'all') return true return staff.category && staff.category.name === selectedCategory })
수정 후:
const filteredStaff = staffMembers .filter(staff => { if (selectedCategory === 'all') return true return staff.category && staff.category.name === selectedCategory }) .sort((a, b) => a.display_order - b.display_order)
✓
Staff page.tsx
일반 모드에서 직원 목록 표시 시 정렬 추가
수정 전:
const filteredStaff = staffMembers.filter(staff => { const matchesSearch = /* search logic */ const matchesCategory = /* category logic */ return matchesSearch && matchesCategory })
수정 후:
const filteredStaff = staffMembers .filter(staff => { const matchesSearch = /* search logic */ const matchesCategory = /* category logic */ return matchesSearch && matchesCategory }) .sort((a, b) => a.display_order - b.display_order)
✓
DragDropFacilityList.tsx
시설 사진 필터링 및 드래그 완료 후 정렬 로직 추가
✓
Facilities page.tsx
filterPhotos 함수에 display_order 기준 정렬 추가
수정 전:
const filterPhotos = () => { let filtered = photos // Filter by search and category... setFilteredPhotos(filtered) }
수정 후:
const filterPhotos = () => { let filtered = photos // Filter by search and category... // Sort by display_order to ensure proper ordering filtered = filtered.sort((a, b) => a.display_order - b.display_order) setFilteredPhotos(filtered) }
